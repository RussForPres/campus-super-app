# Campus Life Super App

## Project Description
Campus Life Super App is a multi-page web application designed to help students manage campus life by providing tools for events, weather updates, study timers, and dining searches.

## Features
- Campus Events page with information cards
- Weather lookup using OpenWeather API
- Study Timer for productivity
- Dining Finder using TheMealDB API

## Technologies Used
- HTML
- CSS
- Bootstrap
- JavaScript
- OpenWeather API
- TheMealDB API

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in a browser OR use Live Server in VS Code
3. Navigate using the menu to explore features

## Deployment
This project is deployed using GitHub Pages.
